# HYSA (مشروع تجريبي)
يحي111
موقع تواصل اجتماعي مصغّر شبيه بتويتر (تسجيل/دخول, نشر, متابعة, إعجاب, بحث) بدون أي مكتبات خارجية.

## التشغيل

1) شغّل السيرفر:

```powershell
node server.js
```

أو وضع التطوير:

```powershell
npm run dev
```

2) افتح المتصفح على:

3000``http://localhost:

ملاحظة: لا تفتح `public/index.html` مباشرة كملف (File) لأن الـ API والملفات قد لا تعمل بشكل صحيح. الأفضل دائما التشغيل عبر السيرفر.

## أين تُحفظ البيانات؟

البيانات تُحفظ في قاعدة بيانات `MongoDB` (عبر `mongoose`) لضمان الاستمرارية على Render.

لازم تضيف متغيرات البيئة التالية:
- `MONGODB_URI`
- `JWT_SECRET`

## ملاحظات مهمة

- الجلسات (Tokens) مؤقتة وتضيع عند إعادة تشغيل السيرفر.
- هذا أساس سريع وليس جاهزا للإنتاج: تحتاج لاحقا إلى قاعدة بيانات حقيقية, صلاحيات/إشراف, حماية أقوى, سجلات, واختبارات.
